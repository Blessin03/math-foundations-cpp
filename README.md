# Math Foundations C++ Toolkit

A C++ console application for practicing foundational math and computing topics, including, finite functions, vectors, matrices, and numerical calculus.

This project was built as an  project to demonstrate modular C++ design, mathematical problem solving, input/output handling.

---

## Features

- Generate truth tables
- Analyze finite functions for range, injectivity, surjectivity, and bijectivity
- Perform vector operations such as addition, subtraction, scalar multiplication, dot product, and magnitude
- Check vector relationships such as same dimension and perpendicularity
- Perform matrix operations such as addition, subtraction, scalar multiplication, transpose, and multiplication
- Estimate derivatives numerically
- Generate numerical limit tables
- Estimate definite integrals using the trapezoidal rule
- Export a  report to a text file

---

## Course Concepts Demonstrated

- Boolean logic
- Functions, domain, codomain, and range
- Injective, surjective, and bijective mappings
- Vectors and vector operations
- Matrices and matrix operations
- Numerical derivatives
- Numerical integration
- Modular programming
- Classes and object-oriented design

- Input validation
